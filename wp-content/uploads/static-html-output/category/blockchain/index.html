程式麻瓜打造區塊鏈網站5–智能合約介紹、編寫環境

L大家都知道區塊鏈有去中心化、不可竄改的特性，藉由這些特點，我們可以利用它來執行需要公平、公正、公開的程式，像是金融、博弈、契約等相關的應用。說到這裡，可能有人會想說既然不可竄改那不就代表程式放上區塊鏈就寫死不會動了嗎?其實應該說是遊戲規則是寫死的，但是參數是可以依據規則改變的，之後實作就知道是怎麼回事了。

智能合約是由叫做solidity的程式語言編寫，可以用線上編輯器remix編寫，也能在VScode裡下載環境編寫，我個人最常用remix編寫，因為可以直接部屬到測試鏈上測試，所以我就以remix來說明拉。

先從左邊的按鈕講起，第一個按鈕式檔案管理區，在這裡可以新增檔案，而solidity檔案的附檔名式.sol

第二個按鈕這區可以選擇 solidity的版本，solidity算是很新的語言所以更新速度很快，且各版本差異都蠻大的，所以編寫的版本需要多留意，而下方的Auto compile選項我建議可以勾選。

第三個按鈕可以選擇環境，javascriptVM虛擬機是指連測試鏈都沒有連上，直接在本地模擬。injected Web3是指透過metamask等工具和區塊鏈互動。web3provider是連到Ganache的虛擬環境。這裡我們只會用到javaVM和injectweb3，在做一般測試的時候使用javaVM因為速度比較快，injectweb3則是部屬合約會用到。

除此之外，我們還要先安裝剛才提到的metamask，metamesk算是個和區塊鏈互動的橋樑，也是存放加密貨幣的錢包，為甚麼要用到錢包?因為我們除了調閱之外的動作如部屬合約以及更改鏈上資料是需要付手續費給礦工的。安裝方法很簡單，進到chrome擴充程式商店下載安裝，照指示創造錢包，若還是有問題可以參考這篇。

安裝且創建完錢包之後，請切換到Ropsten test network，並且到水龍頭領取測試代幣，當然如果你錢很多想直接用主鏈也是可以的^_^

完成上述步驟後，編寫環境差不多都熟悉了，下一篇我們開始了解語法。

程式麻瓜打造區塊鏈網站4-超強大chrome編輯網頁小工具VisBug

在排版網頁的時候每次都要一直調位置座標超級麻煩(應該是我學藝不精)，還好我發現一個chrome extention VisBug 能以滑鼠直接移動位置。

用法很簡單，首先將你要修改的網頁用chrome開啟，點擊visbug圖示，在左邊的欄位選擇第十個看起來像是方向鍵的圖示，再以拖移的方式調整你要修改的區域

再點visbug圖示，把它關掉，在空白處按右鍵，點檢查，到的標籤上按右鍵複製html程式碼，之後回到編輯器把整個程式碼覆蓋就完成了

當然visbug還有很多功能像是打文字、換顏色之類的，甚至可以直接別人的網頁來修改，建議去實做看看，它可以幫助你更了解整個網頁程式的各種功能。

廢話不多，我們直接實作。

首先請在隨便一個地方如桌面按右鍵>

新增資料夾>

在資料夾內再按右鍵新增文字文件>

重新命名，如改成index.html ，index是指網站的主頁，在當網址被點下的時候瀏覽器會讀取你資料夾，然後開啟index.html這個檔案

這裡需特別注意.html這東西叫做副檔名如果不會修改請搜尋”更改副檔名”

改完副檔名後直接點開會發現她會從瀏覽器開啟而不是記事本，這樣就可以算是一個網頁了，所以說網頁的概念很簡單，就是在電腦建立一個資料夾，放入網頁內容讓人來造訪，但是現在網頁沒上伺服器，只有你能造訪。

接下來就是編寫網頁了，要編寫網頁，需要用編輯器(IDE)來編輯，如上圖OPEN選項有顯示的編輯器有記事本、visual studio code，我推薦使用visual studio code來編寫。
